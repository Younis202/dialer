# دليل تشغيل DIALR — خطوة بخطوة

مرحبًا! تطبيق **DIALR** جاهز ويشتغل. هذا الدليل يعلّمك كيف تربطه برقم أمريكي حقيقي وتبدأ تتصل بأي مكان في العالم بأرخص سعر ممكن.

---

## الخطوة 1: تجربة المكالمات المجانية (DIALR Network) — بدون أي إعدادات

التطبيق يدعم مكالمات **P2P مجانية تمامًا** بين أي جهازين يفتحان DIALR — بدون أي رقم، بدون أي مزود.

1. افتح التطبيق في تبويب (Tab) أول.
2. ادخل صفحة **DIALR Network** من القائمة الجانبية.
3. اِنسخ **handle** الخاص بك (مثلاً `DIALR-L3KZRV`).
4. افتح التطبيق في تبويب ثاني (أو على جهاز ثاني).
5. هتشوف الـ handle الأول ظاهر في القائمة.
6. اضغط عليه — مكالمة فيديو/صوت عالية الجودة بدون أي تكلفة.

> هذا يثبت أن WebRTC شغّال، الميكروفون شغّال، وكل شيء جاهز.

## الخطوة 2: إنشاء حساب Voip.ms (لمكالمات العالم بأرخص سعر)

نختار **Voip.ms** لأنها:
- أرخص مزوّد للمكالمات الدولية من رقم أمريكي (أرخص حتى من Twilio بـ 30-50%).
- تدعم **WebRTC مباشرةً** عبر `wss://atlanta1.voip.ms:443` — يعني نتصل من المتصفح بدون أي backend وسيط.
- بتوفّر أرقام أمريكية (DID) مقابل ~0.85$/شهر.

### إنشاء الحساب:

1. روح على [voip.ms/signup](https://voip.ms/signup) وأنشئ حساب جديد.
2. أكّد بريدك الإلكتروني.
3. ادخل **Customer Portal** → اشحن رصيد البداية ($10 يكفي لتجربة).

### الحصول على رقم أمريكي (DID):

1. من القائمة: **DID Numbers → Order DID(s)**.
2. اختار **USA**، اختار **Per Minute** (أرخص خيار)، واختار أي state (مثلاً `California`) أو `Toll Free`.
3. اضغط **Order**. هتدفع ~$0.85 شهريًا + ~$0.009 للدقيقة الواردة.

### إنشاء Sub Account (للاتصال من DIALR):

1. من القائمة: **Sub Accounts → Create Sub Account**.
2. **Username**: مثلاً `dialr-web` (Voip.ms بتزوّد رقمك تلقائيًا قبله، فيصير `123456_dialr-web`).
3. **Password**: اختار باسورد قوي.
4. **Authentication Type**: اختار `User/Password Authentication`.
5. **Device Type**: `IP Authenticated/Locked`.
6. **Internal Extension**: اتركه فاضي.
7. **Callerid Number**: حط رقم الـ DID اللي اشتريته فوق.
8. احفظ.

### تفعيل SMS (اختياري — لو حابب ترسل رسائل):

1. **DID Numbers** → اضغط على رقمك → علّم **Enable SMS / MMS**.

### تفعيل API (لإرسال SMS من التطبيق):

1. **Main Menu → API Configuration**.
2. ضع باسورد API (غير باسورد الحساب).
3. أضف عنوان IP الخاص بـ Replit deployment. (للاختبار المحلي حط `0.0.0.0/0` ثم قيّده لاحقًا.)
4. فعّل API.

## الخطوة 3: ربط Voip.ms بـ DIALR

افتح صفحة **Settings** في التطبيق ثم املأ:

### Identity
- **My Number**: رقمك الأمريكي بصيغة E.164 — مثلاً `+16692222638`
- **Display Name**: اسمك اللي يظهر للمتصل بهم.

### SIP / Voip.ms
- **SIP URI**: `sip:123456_dialr-web@atlanta1.voip.ms` (استبدل الرقم باسم Sub Account اللي عملته).
- **Password**: باسورد الـ Sub Account.
- **WSS Server**: اتركه `wss://atlanta1.voip.ms:443` (أو غيّره لأقرب POP — `chicago2`، `dallas2`، `seattle2`، `losangeles1`، `frankfurt`، `london`...).

### Voip.ms API (للـ SMS)
- **API User**: ايميل تسجيلك في voip.ms.
- **API Password**: باسورد API (مش باسورد الحساب).
- **DID**: رقمك الأمريكي بدون `+` ولا مسافات — مثلاً `16692222638`.

اضغط **Save All**.

## الخطوة 4: تجربة مكالمة دولية حقيقية

1. ارجع لصفحة **Dialer**.
2. اكتب أي رقم دولي بصيغة كاملة، مثلاً `+201001234567` (مصر).
3. التطبيق هيعرض لك:
   - الدولة + العلم + نوع الرقم (موبايل/أرضي).
   - **Cost Optimizer**: مقارنة سعر الدقيقة عبر كل مزود (Voip.ms، Twilio، Telnyx، Plivo) ويبرز الأرخص.
4. اضغط زرار الاتصال الأخضر.
5. الواجهة الكبيرة هتفتح مع timer، ميوت، hold، DTMF، waveform مباشر.

## الخطوة 5: استكشاف بقية الميزات

| الصفحة | استخدامها |
|---|---|
| **Power Dialer** | يطلب ليست كاملة من جهات الاتصال أوتوماتيكيًا، واحد ورا الثاني. |
| **Contacts** | إضافة، استيراد CSV، تصدير CSV، بحث. |
| **Lists** | تجميع جهات اتصال (VIP / Leads / Customers). |
| **History** | كل المكالمات: مدة، تكلفة، تسجيل، export CSV. |
| **Messages** | إرسال/استقبال SMS عبر Voip.ms. |
| **Scheduled** | جدولة مكالمات لتذكيرك تتصل في وقت معين. |
| **Scripts** | قوالب جاهزة بمتغيرات `{{name}}` للاستخدام أثناء المكالمة. |
| **Voicemail Drop** | تسجيل voicemails مسبقًا وتشغيلها لمّا الرد يكون آلي. |
| **Analytics** | charts: مكالمات/يوم، تكلفة/يوم، أعلى دول، توزيع المزوّدين. |
| **DNC** | قائمة أرقام محظورة من الاتصال (للأمان والامتثال). |
| **World Map** | خريطة الدول اللي وصلت لها. |

```bash

npm start          # تشغيل الإنتاج
npm run db:push    # تحديث schema قاعدة البيانات
npm run db:studio  # فتح Drizzle Studio (واجهة DB)
npm run typecheck  # فحص TypeScript
```

## النشر (Deployment)

ضغط **Publish** في Replit واختار **Reserved VM** (مهم — لأن السيرفر فيه WebSocket يحتاج يبقى ثابت).

- Build command: `npm run build`
- Start command: `npm start`
- Port: `5000`

بعد النشر، خد الـ URL واستخدمه من أي مكان في العالم. كل المكالمات هتتم من المتصفح مباشرة لـ Voip.ms — بدون mark-up من جانبنا.

---

## ملاحظات أمان مهمة

- المكالمات الـ P2P **مشفّرة DTLS-SRTP** افتراضيًا (WebRTC).
- مكالمات SIP عبر **WSS (TLS)** — السمة الافتراضية في إعدادات Voip.ms.
- بيانات الاعتماد محفوظة في DB كنص، فلا تشارك URL النشر مع غرباء بدون auth (يمكن إضافتها لاحقًا).
- التزم بقوانين تسجيل المكالمات في بلدك — التسجيل مغلق افتراضيًا.
